How-to: Amazon S3 Multi-User

Tootr comes with a small server that helps share an Amazon S3 bucket with many users. This can be handy for providing space to toot for friends, family, or colleagues. The server brokers write access to the bucket in conjunction with Amazon Web Identity Federation.
